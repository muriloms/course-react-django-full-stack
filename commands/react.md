
# React

- Ambiente

  - Instalar node.js

  - Verificar 
  
  ```node --version```
  
  ```npm --version```
  
- Criar app

```npx create-react-app first-app```


- Iniciar local server

```npm start```

- Router

```npm install react-router-dom --save```


- Style Components

https://styled-components.com/

```npm install --save styled-components```


- React Cooks

```npm install react-cookie```
